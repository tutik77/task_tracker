# Йоу

Весь функционал реализован, структура проекта чистая и масштабируемая, написал минимальные тесты, добавил миграции. 

## Запуск проекта

```bash
# Запуск баз данных
docker-compose up -d

# Создание .env файла по примеру env.example
cp env.example .env

# Запуск приложения
uv run uvicorn main:app --host 0.0.0.0 --port 8000
```
